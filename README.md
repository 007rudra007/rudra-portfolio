# Rudra Trivedi — Personal Portfolio

A premium, fully responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript.

## 🚀 Live Demo
Deploy to Vercel or Netlify — zero configuration needed.

## ✨ Features
- Dark-mode glassmorphism design with animated gradient orbs
- Typewriter hero animation cycling through skills
- Scroll-triggered reveal animations with stagger
- Animated cursor glow following mouse
- Infinite-scrolling certifications marquee
- Mobile-responsive with hamburger nav
- Contact form with mailto fallback
- Active nav highlighting via IntersectionObserver

## 📋 Sections
1. **Hero** — Name, typewriter tagline, stats strip
2. **About** — Bio, links, info cards
3. **Skills** — Categorized chips + scrolling cert strip
4. **Projects** — Hangar, Dynamic Volatility Estimator, Tārā Algorithm
5. **Contact** — Info cards + contact form

## 🌐 Deploy to Vercel
```bash
npx vercel --prod
```

## 🌐 Deploy to Netlify
Drag-and-drop the `portfolio/` folder at [netlify.com/drop](https://app.netlify.com/drop)

## 🛠 Stack
- HTML5 + Semantic markup
- Vanilla CSS (custom properties, grid, flex, animations)
- Vanilla JavaScript (IntersectionObserver, typewriter, mailto)
- Google Fonts: Inter, Space Grotesk, JetBrains Mono

## 📁 Structure
```
portfolio/
├── index.html      # Main markup
├── style.css       # All styles
├── main.js         # All interactivity
├── vercel.json     # Vercel config
├── _redirects      # Netlify config
└── README.md
```
